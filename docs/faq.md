# FAQ

**How do I install STRmie-HD quickly?**  
Use:
```bash
git clone https://github.com/mazzalab/STRmie-HD.git
cd STRmie-HD
conda env create -f STRmie.yml
conda activate STRmie
pip install -e .
```
